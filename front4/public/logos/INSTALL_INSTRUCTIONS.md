# Инструкция по установке официальных логотипов Росатома

## Шаг 1: Скачайте файлы с Яндекс Диска

Перейдите по ссылке: https://disk.360.yandex.ru/d/GPL1mNRL31UCNw

Скачайте следующие папки/файлы:
- **2.1. Логотип** - все файлы из этой папки
- **2.2. Паттерн** - файлы паттернов (если нужны)
- **2.3. Цвета фирменные** - файлы с цветовой палитрой
- **2.4. Шрифты фирменные** - файлы шрифтов (если нужны)

## Шаг 2: Разместите файлы

### Логотипы

Поместите файлы логотипов в следующие директории:

#### Для полного логотипа (с текстом "РОСАТОМ"):
- Цветной вариант → `public/logos/rosatom-logo-full.svg` (или соответствующий формат)
- Белый вариант → `public/logos/rosatom-logo-white.svg`

#### Для символа (только атом):
- Цветной вариант → `public/logos/rosatom-logo-symbol.svg`
- Белый вариант → `public/logos/rosatom-logo-symbol-white.svg`

### Поддерживаемые форматы:
- SVG (предпочтительно)
- PNG (высокое разрешение, минимум 2x)
- PDF (можно конвертировать в SVG/PNG)

## Шаг 3: Обновите компонент (если нужно)

После размещения файлов, компонент `RosatomLogo` автоматически будет использовать их при установке пропа `useOfficial={true}`:

```jsx
// Использование официальных файлов
<RosatomLogo useOfficial={true} height={32} />
<RosatomLogo useOfficial={true} variant="white" height={32} />
<RosatomLogo useOfficial={true} variant="symbol" height={40} />
```

## Шаг 4: Проверка

После размещения файлов проверьте:
1. Файлы имеют правильные имена
2. Файлы доступны по путям `/logos/rosatom-logo-*.svg`
3. Логотипы отображаются корректно на странице

## Примечания

- Если файлы имеют другие имена, переименуйте их согласно инструкции выше
- Если используются форматы PNG/PDF, обновите пути в компоненте `RosatomLogo.jsx`
- Убедитесь, что файлы оптимизированы для веба (размер файла)
