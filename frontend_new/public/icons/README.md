# Иконки и значки

Эта директория предназначена для хранения иконок и значков из брендбука Росатома.

## Извлечение иконок из PDF

Используйте скрипт `extract_logos.py` для извлечения всех изображений из PDF брендбука, затем вручную выберите нужные иконки и сохраните их здесь.

## Форматы

Рекомендуемые форматы:
- SVG (предпочтительно для векторных иконок)
- PNG (для растровых изображений, минимум 2x разрешение)
- WebP (для оптимизации размера)

## Использование в компонентах

```jsx
import icon from '/icons/icon-name.svg';

<img src={icon} alt="Icon" className="w-6 h-6" />
```

## Структура

Рекомендуется организовать иконки по категориям:
- `social/` - социальные иконки
- `navigation/` - навигационные иконки
- `actions/` - иконки действий
- `categories/` - иконки категорий
