# Быстрая установка официальных логотипов

## Вариант 1: Автоматическая установка (рекомендуется)

1. Скачайте файлы с Яндекс Диска: https://disk.360.yandex.ru/d/GPL1mNRL31UCNw
2. Распакуйте архив в любую папку (например, `~/Downloads/rosatom-logos`)
3. Запустите скрипт:
   ```bash
   ./scripts/install-logos.sh ~/Downloads/rosatom-logos
   ```

Скрипт автоматически найдет и скопирует нужные файлы в `public/logos/`

## Вариант 2: Ручная установка

1. Скачайте файлы с Яндекс Диска: https://disk.360.yandex.ru/d/GPL1mNRL31UCNw
2. Найдите файлы логотипов в папке **2.1. Логотип**
3. Скопируйте файлы в `public/logos/` с такими именами:
   - Полный логотип (цветной) → `rosatom-logo-full.svg` (или `.png`)
   - Полный логотип (белый) → `rosatom-logo-white.svg` (или `.png`)
   - Символ (цветной) → `rosatom-logo-symbol.svg` (или `.png`)
   - Символ (белый) → `rosatom-logo-symbol-white.svg` (или `.png`)

## Использование

После установки файлов, обновите компоненты для использования официальных логотипов:

```jsx
// В Header.jsx, Footer.jsx и других компонентах
<RosatomLogo useOfficial={true} height={32} />
<RosatomLogo useOfficial={true} variant="white" height={32} />
```

Или обновите все компоненты сразу, заменив все `<RosatomLogo` на `<RosatomLogo useOfficial={true}`

## Проверка

После установки проверьте:
1. Файлы находятся в `public/logos/`
2. Файлы имеют правильные имена
3. Логотипы отображаются на странице

Если логотипы не отображаются:
- Проверьте имена файлов
- Убедитесь, что файлы в формате SVG или PNG
- Проверьте консоль браузера на ошибки загрузки
