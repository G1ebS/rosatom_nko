# Настройка бесплатной AI-рекомендательной системы

## Быстрый старт

### Вариант 1: Hugging Face Inference API (Рекомендуется)

1. **Регистрация**: Перейдите на https://huggingface.co/settings/tokens
2. **Создайте токен**: Нажмите "New token" и скопируйте токен
3. **Добавьте в .env**: 
   ```
   REACT_APP_HUGGINGFACE_API_KEY=ваш_токен_здесь
   ```
4. **Готово!** Система автоматически использует AI-рекомендации

**Лимиты**: До 1000 запросов в месяц бесплатно

### Вариант 2: Без API (Локальный алгоритм)

Если не хотите использовать API, система автоматически использует локальный алгоритм на основе правил. Никаких настроек не требуется!

## Использование

Система автоматически:
1. Анализирует интересы пользователя
2. Сравнивает с описаниями НКО
3. Выдаёт топ-5 рекомендаций

## Код уже готов

Файл `src/utils/aiRecommendations.js` содержит:
- `getAIRecommendations()` - с использованием Hugging Face API
- `getSimpleRecommendations()` - простой алгоритм (fallback)
- `getRuleBasedRecommendations()` - алгоритм на основе правил

## Интеграция в компонент

Компонент `AIRecommendations` уже создан и используется на главной странице. Он автоматически:
- Пробует использовать AI API
- Если не получается - использует простой алгоритм
- Показывает загрузку и ошибки

## Тестирование

1. Войдите в систему (user@example.com / password)
2. Перейдите на главную страницу
3. Прокрутите до секции "Персональные рекомендации"
4. Увидите рекомендации на основе ваших интересов
